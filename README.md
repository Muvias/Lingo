# Lingo

Lingo é uma réplica simples do Duolingo, aplicativo para aprender novos idiomas de uma maneiras simples e divertida, com missões, pontos e ranqueamento.

## Ferramentas utilizadas:

<ul>
  <li>NextJS 14</li>
  <li>TypeScript</li>
  <li>TailwindCSS</li>
  <li>Clerk</li>
  <li>Zustand</li>
  <li>NeonDB</li>
  <li>Drizzle Orm</li>
  <li>Stripe</li>
</ul>
